# ***AMANDA BOUTIQUE***
#### **Video Demo:** https://youtu.be/XqdgIVWoqEk
### **Description:** *Amanda Boutique* is a Python project using the Django framework to create a showroom for a designer of custom-made quinceañera dresses. It also manages the databases for Catalogo, Clientes, Proveedores and PerfilUsuarios

**Technologies**

- Python
- Django
- HTML
- SQLite

**File and Folder Description:**  

**AmandaProjecto** is the main Django project where the following files were used:  

- **Setting.py**: Project configuration.  

- **Urls.py**: Project URLs.

**BoutiqueApp** is the app that displays the showroom's main page, with a dynamic section that shows the dresses in the Catalog and manages the Catalog database.

- **views.py**: Python logic.

- **forms.py**: Python logic for database formats.

- **models.py**: Catalog database definition.

- **urls.py**: URLs for the BoutiqueApp.

- **Templates folder**: Contains all the HTML files used to manage the Catalog database.

- **index.html**: Showroom's main page, displaying the dynamic catalog, the about Amanda section, Contacts, and the administration area for the different databases.

**ClientesApp** is the app that manages the customer database.

- **views.py**: Python logic.

- **forms.py**: Python logic for database formats.

- **models.py**: Customer database definition.

- **urls.py**: URLs for the ClientesApp.

- **Templates folder**: Contains all the HTML files used to manage the Customers database.

**ProveedoresApp** is the app that manages the Suppliers database.

- **views.py**: Python logic.

- **forms.py**: Python logic for the database formats.

- **models.py**: Definition of the Suppliers database.
- **urls.py**: URLs of the Suppliers app.

- **Templates folder**: where each of the HTML files used to manage the Suppliers database are located.

**LoginApp** is the app that manages the User Profile database.

- **views.py**: Python logic.

- **forms.py**: Python logic for the database formats.

- **models.py**: Definition of the User Profile database.

- **urls.py**: URLs of the Suppliers app.

- **Templates folder**: where each of the HTML files used to manage the Suppliers database are located.

**Usage:**
- The Home button in the navbar will always take you to the home page.
- The About Amanda button will take you to a descriptive section about Amanda Boutique.
- The Contact button will take you to the contact information.
- The Admin button will take you to the section where the different databases are created and listed.
- If the user is not logged in, the Login and Register options will appear in the navbar. If they are logged in, the Login and Logout options will appear.
• In the Administration Panel, you will find the following buttons:
- Administration: This will take you to the administration of users, groups, and databases.
- There are also three buttons for uploading information to the Catalog, Customers, and Suppliers databases.
- To manage these databases, you must be logged in.
